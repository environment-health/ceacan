2 Methods

2.1 Data

2.1.1 Canadian communities

2.1.1.1 Geolocated placenames

The geolocated placenmes dataset provides the coordinates of all named places across Canada, including cities, villages, First Nations Communities, etc (Science and Innovation, 2020). This dataset contains 10676 geolocated placenames across all of Canada. We use these locations as a proxy of Canadian communities locations.

2.1.1.2 First Nations location

The First Nations location dataset was used to identify the locations of First Nations across Canada (Canada, 2022a). This dataset contains the location of 637 First Nations communities. While the locations of the First Nations communities in Canada are included in the geolocated placenames dataset, we used this dataset to be certain that the names of the First Nations were appropriate. We identified and removed the coordinates in the geolocated placemanes dataset that were closest to each First Nations locations; we then merged the First Nations locations with the geolocated placenames to form a single dataset.

2.1.1.3 Inuit Communities location

The Inuit Communities location dataset was used to identify the locations of Inuit communities across Canada (Relations and orthern Affairs Canada, 2020). This dataset contains the location of 53 Inuit Communities. As with the First Nations location dataset, the Inuit Communities locations are included in the geolocated placenames, yet we used this dataset to be certain of Inuit Community names. The same approach was used to integrate both datasets together (see previous section).

2.1.2 Social vulnerabilities

So far, we characterized social vulnerabilities using three main sources of data to obtain the following list:

  • Proximity to employment
  • Proximity to grocery stores
  • Proximity to pharmacies
  • Proximity to health care
  • Proximity to child care
  • Proximity to primary education
  • Proximity to secondary education
  • Proximity to public transit
  • Proximity to neighborhood parks
  • Proximity to libraries
  • Dwelling condition
  • Housing suitability
  • Acceptable housing
  • Proximity to waste treatment and disposal facilities
  • Proximity to water and wastewater systems

Below is a brief description of the data and how the were processes.

2.1.2.1 Proximity measures database

The proximity measures database is the result of a collaboration between Statistics Canada and Canada Mortgage and Housing Corporation that aimed at implementing a set of proximity measures to services and amenities across Canada (Canada, 2020b, 2020a). The scale of the assessment is at dissemination block level from Statistics Canada, which results in ~500 000 records. In essence, the proximity measurements assess the distance of each diseamination block with each amenities and services considered (see here for more details on the methods). Each measurement is normalized between 0 and 1, 0 being the closest and 1 the farthest from a service or amenity. The proximity measures included are proximities to:

  • employment
  • grocery stores
  • pharmacies
  • health care
  • child care
  • primary education
  • secondary education
  • public transit
  • neighborhood parks
  • libraries

For our work, we are using those proximity measurements as a proxy of social vulnerability for communities located in each dissemintation block. We thus intersected each canadian community with the proximity measures database to extract the value of proximities. For the first iteration of our work, we are considering all 10 proximity measures available in the database.

2.1.2.2 Housing statistics

We also characterized the social vulnerability of canadian communities based on housing conditions. Those data are available through Statistics Canada censuses. More specifically, we used dwelling condition (Canada, 2022d), housing suitability (Canada, 2022e) and acceptable housing (Canada, 2022f) data available from the 2021 Canadian census at the census subdivision scale (Canada, 2022c, 2022b).

The dwelling condition identified, among other things, the number of houses that are in need of major repairs in each subdivision census. Housing suitability, meanwhile, evaluates the suitability of a house as a function of its capacity and the number of occupants to provide an assessment of the number of houses where household size is not suitable in each census subdividion. Finally, acceptable housing is evaluated as a function of adequacy (i.e. house not requiring any major repars), affordability (i.e. shelter costs less than 30% of total before-tax household income), and suitability (i.e. house has enough bedrooms for the size and composition of household) thresholds. We used number of houses that fell below any of the acceptabiliy thresholds in census subdivision as a measure of social vulnerability.

For each of the housing statistics considered, we evaluated the percent of socially vulnerable houses as the intensity of social vulnerability within each census subdivision. We then intersected each Canadian community with census subdivisions to identify the intensity of housing vulnerability to which Canadian communities were exposed.

2.1.2.3 Water and wastewater systems

We characterized accessibility to water and wastewater systems using the National Pollutant Release Inventory (NPRI) (Environment and limate Change Canada, 2022). The NPRI dataset provides an inventory of pollutant releases, disposals and transfers for recycling between 1993 and 2020. The dataset contains the locations of all reporting facilities and the types of sectors to which they belong. We extracted all reporting facilities that belonged 1) to Waste Treatment and Disposal and 2) to Water and Wastewater Systems to identify the locations of such facilities across Canada. We then measured the distance of each Canadian community to the locations of reporting facilities. Canadian communities located beyond 250 km of a facility were given the value of 250 to cap the maximum distance considered. All distances were then normalized by dividing by 250 to obtain an index between 0 and 1, with Canadian communities closest to water and wastewater systems receiving a value of 1, and those farthest a value of 1.

2.1.3 COVID-19 data

The COVID-19 data used comes from an initiative called the Timeline of COVID-19 in Canada (CovidTimelineCanada) (Berry et al., 2021). This data repository is the result of the integration of multiple datasets nationally and provide data that is updated daily for all of Canada. The data are available at the scales of Canada, of the provinces/territories, and of the health regions. Here, we used the data at the health region scale, which is provided for daily cases and deaths from 2020-01-24 to today. For more information, visit the CovidTimelineCanada GitHub repository.

For now, we used cumulative cases and deaths in each health region to evaluate the severity of the pandemic. We then intersected each Canadian community with the COVID-19 health regions to evaluate the severity of the pandemic to which each community was exposed to since the beginning of the outbreak.

2.2 Data transformation

All social vulnerabilities considered are relative and range from 0 to 1. A value of 0 means that a community is least vunerable, while a value of 1 means that a community is most vulnerable compared to other communities across the country.
